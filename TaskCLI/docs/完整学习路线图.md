## 阶段一：基础（1-2个月）

### 1.1 Rust 基础（2周）

**目标**：掌握 Rust 基本语法，能写简单程序

**学习内容**：
- 变量和基本类型
- 函数定义
- 流程控制（if, loop, match）
- 所有权基础

**练习**：
- 写一个计算器
- 实现冒泡排序

**资源**：
- [The Rust Book 前6章](https://doc.rust-lang.org/book/)

---

### 1.2 C++ 基础（2周）

**目标**：掌握 C++ 基本语法，能写简单程序

**学习内容**：
- 变量和基本类型
- 函数和类
- 流程控制
- 构造函数和析构函数

**练习**：
- 写一个学生管理系统
- 实现计算器

**资源**：
- [learncpp.com 前10章](https://www.learncpp.com/)

---

### 1.3 开发工具（1周）

**目标**：熟练使用开发环境

**学习内容**：
- VS Code / CLion 配置
- Rust: cargo, rustc, rust-analyzer
- C++: CMake, make, g++
- Git 版本控制

**练习**：
- 用 Git 管理 TaskCLI 项目

---

## 阶段二：核心技能（2-3个月）

### 2.1 Rust 所有权系统（2周）⭐

**目标**：深入理解 Rust 核心特性

**学习内容**：
- 所有权和借用规则
- 生命周期标注
- 借用检查器原理

**关键问题**：
- 为什么 Rust 不需要 GC？
- 悬挂引用是什么？

**项目**：
- 实现自己的 Vec<T>

---

### 2.2 Rust 高级特性（2周）

**目标**：掌握 trait、泛型、错误处理

**学习内容**：
- Trait 系统和 trait object
- 泛型编程
- Result 和 ? 操作符
- 智能指针（Box, Rc, Arc）

**项目**：
- 扩展 TaskCLI 功能

---

### 2.3 C++ 进阶（2周）

**目标**：掌握现代 C++ 特性

**学习内容**：
- 智能指针
- RAII 惯用法
- 移动语义
- Lambda 表达式

**项目**：
- 用智能指针重写 TaskCLI

---

### 2.4 并发编程（2周）

**目标**：掌握多线程编程

**学习内容**：
- Rust: thread, Mutex, Channel, Arc
- C++: thread, mutex, future, atomic
- 线程安全原理
- 内存模型

**项目**：
- 并发文件下载器

---

### 2.5 C++ 模板编程（2周）

**目标**：掌握模板元编程

**学习内容**：
- 函数模板和类模板
- 模板特化
- 变长模板
- Type traits

**项目**：
- 实现通用容器库

---

**目标**：实现 HTTP 服务器

**技术栈**：

- C++: libuv, custom

- 静态文件服务
- JSON API

**项目地址**：

- REST API 服务器

**目标**：实现简单数据库

**技术栈**：
- Rust: tokio-postgres
- C++: SQLite

**功能**：
- SQL 解析（简化版）
- B+Tree 索引
- 事务处理
- 连接池

---

**学习内容**：
