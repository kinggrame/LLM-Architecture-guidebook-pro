### 1.5 变量与可变性

Rust 的变量默认是不可变的，这是为了安全：

```rust
fn main() {

    let mut y = 6;
    y = 7;
    println!("y = {}", y);
    
    // 常量（必须标注类型）
```

**关键概念：**
- `let`: 声明变量

- `const`: 声明常量（必须标注类型）

### 2.1 标量类型

Rust 有四种标量类型：

```rust
fn main() {
    // 整数
    let i: i32 = 42;          // 有符号 32 位
    let u: u32 = 100;         // 无符号 32 位

    let usize: usize = 10;    // 平台相关无符号
    
    // 浮点数
    let f64_val: f64 = 3.14;  // 64 位浮点
    let f32_val: f32 = 3.14f32; // 32 位浮点

    println!("浮点: {}, {}", f64_val, f32_val);
    println!("布尔: {}", b);
    println!("字符: {}, {}", c, emoji);
}
```

**整数范围：**

```rust
    if let Some(c) = first_char("hello") {

}
```

### 3.3 循环

**loop - 无限循环：**

```rust
fn main() {
    let mut count = 0;
    
    loop {
        println!("count = {}", count);
        count += 1;
        
        if count >= 5 {
            break;  // 退出循环
        }
    }
    
    // 带标签的循环
    let mut i = 0;
    'outer: loop {
        let mut j = 0;
        loop {
            if i == 2 && j == 2 {
                break 'outer;  // 退出外层循环
            }
            j += 1;
            if j >= 3 { break; }
        }
        i += 1;
    }
    println!("Stopped at i={}, j={}", i, j);
}
```

## Day 4: 所有权系统（核心！）

### 这是 Rust 最独特的概念，必须深入理解！

### 4.1 什么是所有权？

Rust 的核心特性：**所有权系统**。它让 Rust 能够在没有垃圾回收的情况下保证内存安全。

```rust
fn main() {
    // 变量 s1 拥有字符串的所有权
    let s1 = String::from("hello");
    
    // s2 获取 s1 的所有权，s1 不再有效！
    let s2 = s1;
    
    // println!("{}", s1);  // 错误！s1 已经无效
    println!("{}", s2);      // 正确
}
```

**为什么需要所有权？**
- **无 GC**：不像 Java/Python，Rust 没有垃圾回收器
- **零成本抽象**：不需要运行时开销
- **内存安全**：在编译时防止内存错误

### 4.2 所有权规则

```
1. Rust 中的每个值都有一个所有者（owner）
2. 每个值同时只能有一个所有者
3. 当所有者离开作用域时，值会被自动丢弃
```

```rust
fn main() {
    {                       // s 在这里无效
        let s = String::from("hello");  // s 生效
        println!("{}", s);
    }                       // s 在这里离开作用域，被释放
    
    // println!("{}", s);  // 错误！
}
```

### 4.3 移动 vs 复制

**对于堆上数据（如 String）：移动**

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1;  // s1 移动到 s2，s1 失效
    
    // println!("{}", s1);  // 编译错误！
    println!("{}", s2);      // 正常
}
```

**对于栈上数据（如 i32）：复制**

```rust
fn main() {
    let x = 5;
    let y = x;  // 复制，x 仍然有效
    
    println!("x = {}, y = {}", x, y);  // 都有效
}
```

**如何选择移动还是复制？**
- 实现了 `Copy` trait 的类型（栈上数据）：复制
- 实现了 `Clone` trait 的类型：可以显式克隆
- 其他类型：移动

```rust
fn main() {
    // 显式克隆
    let s1 = String::from("hello");
    let s2 = s1.clone();  // 克隆，两个都有效
    
    println!("s1 = {}, s2 = {}", s1, s2);  // 都可以用
}
```

### 4.4 引用与借用

不想转移所有权？使用**引用**：

```rust
fn main() {
    let s1 = String::from("hello");
    
    // & 表示引用（借用）
    let len = calculate_length(&s1);
    
    println!("'{}' 的长度是 {}", s1, len);  // s1 仍然有效
}

// 参数是引用，不获取所有权
fn calculate_length(s: &String) -> usize {
    s.len()
}  // s 离开作用域，但不丢弃数据
```

**借用规则：**
1. **不可变引用**：可以有很多个 `&T`
2. **可变引用**：只能有一个 `&mut T`，且不能同时有不可变引用

```rust
fn main() {
    let mut s = String::from("hello");
    
    // 不可变引用
    let r1 = &s;
    let r2 = &s;
    println!("{} and {}", r1, r2);
    
    // 不可变引用使用完毕后，才能创建可变引用
    let r3 = &mut s;
    println!("{}", r3);
}
```

### 4.5 生命周期（初步了解）

当函数返回引用时，需要标注生命周期：

```rust
fn main() {
    let string1 = String::from("long string is long");
    let result;
    {
        let string2 = String::from("xyz");
        result = longest(&string1, &string2);
    }
    println!("The longest string is '{}'", result);
}

// 生命周期参数 'a 告诉编译器引用的有效范围
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() {
        x
    } else {
        y
    }
}
```

```rust
struct User {
    username: String,
    email: String,
    active: bool,
    sign_in_count: u64,
}

fn main() {
    // 创建结构体实例
    let user1 = User {
        email: String::from("someone@example.com"),
        username: String::from("someusername123"),
        active: true,
        sign_in_count: 1,
    };
    
    // 访问字段
    println!("User: {}", user1.username);
    println!("Email: {}", user1.email);

    let mut user2 = User {
        email: String::from("another@example.com"),
        username: String::from("anotherusername"),
        active: true,
        sign_in_count: 1,
    };
    
    user2.email = String::from("newemail@example.com");
    println!("New email: {}", user2.email);
}
```

```rust
fn main() {
```

### 5.4 方法（impl）

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    // 构造函数（Rust 没有内置构造函数，用普通函数）
    fn new(width: u32, height: u32) -> Self {
        Self { width, height }
    }
    
    // &self 表示对结构体的不可变引用
    fn area(&self) -> u32 {
        self.width * self.height
    }
    
    // 可变方法
    fn set_width(&mut self, width: u32) {
        self.width = width;
    }
    
    // 静态方法（不需要 self）
    fn square(size: u32) -> Self {
        Self { width: size, height: size }
    }
}

fn main() {
    let rect = Rectangle::new(30, 50);
    println!("Area: {} square pixels", rect.area());
    
    let square = Rectangle::square(10);
    println!("Square area: {}", square.area());
}
```

### 5.5 方法中的所有权

```rust
struct StringHolder {
    value: String,
}

impl StringHolder {
    // 消耗所有权（获取 self）
    fn consume(self) -> String {
        self.value
    }
    
    // 借用（获取 &self）
    fn len(&self) -> usize {
        self.value.len()
    }
    
    // 可变借用（获取 &mut self）
    fn modify(&mut self, new_value: String) {
        self.value = new_value;
    }
    
    // 借用特定字段
    fn get_string(&self) -> &String {
        &self.value
    }
}

fn main() {
    let holder = StringHolder { 
        value: String::from("hello") 
    };
    
    println!("Length: {}", holder.len());
    
    let mut holder2 = holder;  // 移动
    // println!("{}", holder.len());  // 错误！holder 已移动
    
    // 消耗所有权
    let s = holder2.consume();
    println!("Consumed: {}", s);
}
```

## Day 6: 错误处理 - Result 与 Option

### 6.1 Option 类型

处理可能为空的值：

```rust
fn main() {
    // 创建一个 Option
    let Some(value) = some_function_that_returns_option();
    
    if let Some(v) = value {
        println!("Got value: {}", v);
    }
    
    // 或者使用 unwrap
    let x = some_function_that_returns_option().unwrap();
}

// 返回 Option<i32>
fn some_function_that_returns_option() -> Option<i32> {
    Some(42)
    // 或者 None
}
```

**Option 的常见用法：**

```rust
fn main() {
    // if let 简化
    let name = get_name();
    if let Some(n) = name {
        println!("Name: {}", n);
    }
    
    // unwrap_or 提供默认值
```
