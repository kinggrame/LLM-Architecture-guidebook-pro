## 🚀 快速开始

### 1. 安装 Rust (前三个阶段)

```bash
# Windows (PowerShell 管理员)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# macOS / Linux
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 2. 安装 Python + PyTorch (第四阶段)

```bash
# 推荐使用 Anaconda
conda install python=3.10

# 创建环境
conda create -n deepllm python=3.10
conda activate deepllm

# 安装 PyTorch (CUDA 11.8)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# 安装项目依赖
pip install -r DeepLLM/requirements.txt
```

## 💡 学习建议

### 高效学习法

1. **先运行，再理解**
   - 先把代码跑起来看到效果
   - 再回头看原理

2. **带着问题学**
   - "为什么需要所有权？"
   - "Attention 为什么这样设计？"
   - "训练和推理有什么区别？"

3. **动手实践**
   - 每天的练习题一定要做
   - 尝试修改代码看效果

4. **画架构图**
   - 帮助理解数据流动
   - 加深记忆
