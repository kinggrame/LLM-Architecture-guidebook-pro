## 快速参考

### 核心概念

| 概念 | 说明 |
|------|------|
| Engine | 主引擎，协调所有子系统 |
| ScriptEngine | 解析和执行剧情脚本 |
| SceneManager | 管理游戏场景切换 |
| ResourceManager | 加载和缓存游戏资源 |
| RenderEngine | 绘制画面到屏幕 |
| Command | 脚本命令（背景、对话等） |
| Scene | 游戏场景（标题、游戏、菜单） |

### 常用命令

```bash
# Rust 项目
cargo build     # 构建
cargo run       # 运行
cargo check     # 检查代码

# 脚本命令
@bg <图片>      # 显示背景
@char <图片>    # 显示角色
@music <音频>   # 播放音乐
@jump <标签>    # 跳转到标签
@end            # 结束游戏
```
