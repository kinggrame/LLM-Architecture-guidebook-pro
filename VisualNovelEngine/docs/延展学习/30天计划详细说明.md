## 🚀 如何使用本计划

### 建议学习顺序

1. **第一轮（Day 1-10）**: 专注 Rust 核心技能
2. **第二轮回（Day 11-20）**: 专注 C++ 核心技能
3. **第三轮（Day 21-30）**: 选择感兴趣的高级主题深入

### 学习方法

1. 每天学习一个主题
2. 运行代码 Demo
3. 完成练习题
4. 尝试修改代码
5. 对比 Rust 和 C++ 的实现
